#### 运行示例
**基于docker构建项目**，
需要安装：
1. docker
2. docker-compose


执行 `docker-compose up` 命令启动项目

**本地编译**

需要安装：
1. go环境
2. gcc 环境

执行 `go mod tidy`  `go build`
